# Raw Data

This directory contains original, unmodified tag data files from CATS and DTAG deployments.

## Important Notes

- **Do not modify** files in this directory
- Large data files are excluded from git (see `.gitignore`)
- Keep original file names for traceability

## File Naming Convention

Suggested naming format:
```
{species}_{individual_id}_{tag_type}_{date}.{ext}

Examples:
- humpback_001_CATS_20150215.csv
- humpback_002_DTAG_20160308.mat
```
